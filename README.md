# 🎮 XPORTS Gear — E-Commerce Web App

A full-featured e-commerce web application built with **React** and **TypeScript**, offering a smooth shopping experience with authentication, multi-language support, dark/light mode, and a fully responsive design.

## 🛠 Tech Stack

- **React** (with TypeScript)
- **React Hooks**
- **React Router DOM v7**
- **Axios**
- **Redux Toolkit** + **React-Redux**
- **Bootstrap**
- **SASS (SCSS)**
- **Supabase** (Database & Authentication)
- Fully responsive, mobile-first design

## 📦 Installation

Clone the repository and install dependencies:
\`\`\`bash
git clone https://github.com/Boyukkhanim/frontend-final-project.git
cd frontend-final-project
bun install
\`\`\`
Run the development server:
\`\`\`bash
bun dev
\`\`\`
Build for production:
\`\`\`bash
bun run build
\`\`\`

## 🗄 Environment Variables

Create a `.env` file in the root directory and add your Supabase credentials:

\`\`\`env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
\`\`\`
